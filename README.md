# IssueHub

IssueHub is a full-featured internal issue tracking web application designed for companies to manage projects, track tasks, and streamline team collaboration. The system provides structured workflows, role-based access control, and real-time issue management to improve productivity and accountability within organizations.

---



##  User Roles

IssueHub supports three levels of users:

### 1️⃣ System Administrator
- Manage users and roles
- Create and manage projects
- Configure system-level settings
- View all issues across projects

### 2️⃣ Project Manager
- Create and assign issues
- Set priority and deadlines
- Manage workflow statuses
- Monitor team performance

### 3️⃣ Employee (Developer / Staff)
- View assigned issues
- Update issue status
- Add comments and attachments
- Track personal tasks
